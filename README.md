# Cloud Hotel Service ( TeamProject 06/15 ~ 07/13)

### 클라우드 호텔 홈페이지 제작

제작의도 : 기존의 복잡하고 사용자에게 제대로 전달되지 못하고 있는 정보, UI, 서비스의 진행방식을 개선하여
         누구나 사용할 수 있는 편의성과 간편화를 갖춘 클라우드 호텔 홈페이지 제안.

제작기간 : 23/06/15 ~ 23/07/13
